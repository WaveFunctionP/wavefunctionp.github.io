AmphetaDesk SideBar

I've been trying in vein to find a adequate news program for windows. There are a few on the market, but few utilize a 3 pane design with a simple interface. Well, that is 'till I found AmphetaDesk. I've used it before, but because it didn't use the requisite 3 window panes, I left in alone to find another solution. I did like AmphetaDesk for it's simplicity and light weight. After a few weeks of using NewzCrawler, I slowly began to dislike many of is quirks and especially that I was going to have to pay for it come a few days. It really didn't justify the cost in my mind.

Fast Forward to yesterday. I reloaded AmphetaDesk and found out you could customize the templates used to generate the headlines page. I've also been hearing a lot of noise on the net about Mozilla sidebars. Couple that with a little JavaScript and viola, 3 pane (sorta) RSS newsreader mozilla sidebar. [screenshot] It's a bad pic, but you get the idea.

The core idea is to use AmphetaDesk to aggregate the feeds, along with it's feed management features, modify the index template to fit in a sidebar and add some collapsible menus to the feed list. Tweak and you've got a totally free, easy to use, way to read news feeds.

All you need to get it running is to download and install AmphetaDesk, copy this index file to your template folder(right click, save target as, rename to index.html), and install the side bar. (The last link is a javascript that installs the AmphetaDesk URI: "http://127.0.0.1:8888/index.html", to your sidebar)

Note. I've removed the excerpt/description container from the perl script because the list was too long for my tastes. If you want it just copy the appropriate container from the default template to the new one.

Remember to back up your default template (rename it, or copy it to another directory) before you over write it with the new template. It really doesn't matter, because you should have the zip file you downloaded that you can restore from, but it might save you some trouble if you've already made some changes to the template that you want to save.

All page formatting is done in DOM, so in other words the sidebar is just a webpage. CSS and scripting all apply the same as if it were in the main window. So, CSS is how I chose to apply style. target="_content" in the "a" tags makes the links display in the main window instead of the side bar. Sidebar information can be found here and an AmphetaDesk template explanation is here. (Don't know perl? Neither do I, all you need to know is all variables start with "$", other than that you'll catch on pretty quick)

Enough of this noise, have fun. Remember to tell me what you think, any suggestions?
