For Science!

This was small mod for Kerbal Space Program. It is designed to automate some of the more tedious elements of the gameplay experience. It was written in C# and uses the MonoDevelop framework, Unity Game Engine API and Kerbal Space Program API.

This was my first mod written from scratch. It has been well recieved with over 15,000 downloads since it was released in May '14. It was also featured on PC Gamer as one of "best Kerbal Space Program mods".
