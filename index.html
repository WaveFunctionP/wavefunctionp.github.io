For Science!

This is mod for Kerbal Space Program. It is designed to automate some of the more tedious elements of the gameplay experience. It was written in C# and uses the Unity Game Engine and Kerbal Space Program APIs.

This was my first mod written from scratch. It has been well recieved with over 15,000 downloads since it was released in May '14. It was also featured on PC Gamer as one of "best Kerbal Space Program mods" and has even been forked for expanded applications by other authors.

SimpleScienceFix

This is a small usability mod for Kerbal Space Program meant to address a small annoyance with the way the game works. It was written in C# and uses the Unity Game Engine and Kerbal Space Program APIs.

It has been fairly well recieved with over 1,000 downloads since May '15.
