More Site Features

As you see, I've have yet another site design. It's nothing major, just a few CSS tweaks. I like the general layout, so I won't be changing that. I just wanted something that was a little easier on the eyes and that still provided a definite structure to the news post without being so rigid looking. The change in fonts helps in readablity, while a change in the hyperlinks' formatting cleans up the text. I standardized the font size; choosing to use a fixed "px" value instead of the more general "normal", "x-small", etc. values. I noticed that the different browsers were displaying the text size much differently. Using a fixed pitch solved the sizing problems, and now you should see this site's text at the size they were intended.

A few people have noted that the text size was too small. That should solve the problem, but to insure that the more visually impaired reader can see the text; I've included a simple script to change the text size of each post either smaller or larger to suit each individual's tastes. You can do this with most browsers in the options, but I some people don't know how to do this. Besides that, it gave me an excuse to use my new found Javascript skills. All you need to do is click the apropriate link on the navbar to change the text size. I'll post the script later in another article for those of you that want to know how write the script.

i just wonder how long this design will last...
